# Modulo 1

How to make this run:

```sh
/* Windows Powershell Edition */
npm install
cd '.\modulo 1\ejercicio 1\'
parcel index.html
```

y a volar. 

notese que el proceso es el mismo para todos los ejercicios [salvo el 4, pues no tiene dependencias.]
